vim comands:
#see: https://www.fprintf.net/vimCheatSheet.html
#examples: http://www.radford.edu/~mhtay/CPSC120/VIM_Editor_Commands.htm
#vimscript: http://learnvimscriptthehardway.stevelosh.com/
i - insertion mode
Esc - exit the insertion mode
:w - save workspace in the file
:wq - save workspace and quit
:q - exit with a warning that unsaved work
:q! - exit without warning

#Flight Manual: http://flight-manual.atom.io
